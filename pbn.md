- Private Blog Network - это частная сеть сайтов, сетка сайтов для продвижения одного сайта
- Тематика у сайтов сетки обычно узкотематическая (лэндинг + блог), также можно:
  - новостные сайты
  - дроп аналогичной тематики и развивать его
  - можно расширять тематику
  - ревьюшники/отзовики
  - мнобренды
  - моно товары
- Ссылаться должна только на основной сайт
- CMS WordPress
- Контент
  - можно восстановить
  - создать самому
  - спины
  - ИИ
  - парсинг
  - копипаст
  - сплоги: бесплатный хостинг типа wordpress, blogspot
- Футпринты: до 5 можно не думать.
- Ссылки:
 - не ставить ссылки с основного
 - не делать перекрестных ссылок
 - 50% на главную, 50% на внутренню
 - 50% с анкором, 50% без анкора
